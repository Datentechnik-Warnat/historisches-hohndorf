Mitmachen beim Projekt „Historisches Hohndorf“

Du hast Informationen, Geschichten oder Bilder zu Hohndorf und möchtest sie mit anderen teilen? Großartig!
Das Projekt lebt vom Mitmachen - und GitHub hilft uns, Änderungen strukturiert und nachvollziehbar einzupflegen. So funktioniert's:

  1. Projekt forken

    Gehe auf das Repository: https://github.com/Datentechnik-Warnat/historisches-hohndorf
    Klicke oben rechts auf "Fork", um eine eigene Kopie des Projekts in deinem GitHub-Account zu erstellen.

  2. Änderungen vornehmen

    Bearbeite in deinem Fork die Inhalte - zum Beispiel:
    • Eine neue Seite anlegen
    • Einen bestehenden Text ergänzen oder korrigieren
    • Ein historisches Foto oder eine Karte hinzufügen

    Du wirst beim Abschluss um eine Nachricht, die "Commit-Nachricht" bzw. Pull-Request-Beschreibung gefragt. Nutze dabei bitte folgendes Format: 
    Ort(Musterstraße 12)

Erbauung der Kirche konkretisiert. Quelle: Ortschronik Band 2, Seite 45.
    Das Stichwort Ort(...) hilft uns, die Änderung einem konkreten Ort zuzuordnen. Der GitHub-Benutzername zeigt automatisch, wer die Änderung vorgeschlagen hat.

  3. Pull Request stellen

    Wenn deine Änderungen fertig sind:
    • Öffne deinen Fork auf GitHub
    • Klicke auf „Pull Request“
    • Starte den Vergleich eines Projektes mit dem Originalprojekt
    • Reiche den Vorschlag zur Prüfung ein

    Die Projektverantwortlichen sichten deine Änderung und schalten sie ggf. frei.
