+++ title = 'Hauptstraße 14' draft = true type = "Page" subtitle = '' weight = 0 geo_lat = 50.746436 geo_lng = 12.670742 +++ Vom neugebauten Feuerwehrhaus hat man einen schönen Blick auf das Seitengebäude des Vierseithofes von Fiedlers, Hauptstraße 14. Es ist ein um 1800 gebautes Wirtschaftsgebäude mit seltener Oberlaube. Rechts im Bild sieht man ein kleines Häuschen, in dem die Gasleitungen kontrolliert werden. Ursprünglich stand daneben auch ein turmartiges Transformatorenhäuschen, das abgebrochen wurde. Im Hintergrund sieht man das im Jahre 1898 erbaute Klinkerhaus vom damaligen Schullehrer Alban Kießling, heute gehört es Familie Scheibner. {{< imagerow >}} {{< image src="images/IMG_9155 9.jpg" height="400" >}} {{< image src="images/IMG_9276.jpg" height="400" >}} {{< /imagerow >}} {{< nav >}}
