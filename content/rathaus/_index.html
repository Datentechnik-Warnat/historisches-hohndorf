+++ title = 'Rathaus' draft = true subtitle = 'Rödlitzer Straße 84' +++ {{< imagerow >}} {{< image src="images/buchprojekt-111.jpg" align="right" height="280" >}} {{< image src="images/IMG_9147-27.jpg" align="right" height="280" >}} {{< image src="images/IMG_4428 27.jpg" align="right" height="280" >}} {{< image src="images/2011-hohndorf-685.jpg" align="right" height="280" title="2011" >}} {{< /imagerow >}} {{< clear >}} {{< timeline/timeline >}} {{% timeline/item to="1873-06-12" %}} Die Grundsteinlegung für das neue Schulgebäude in Hohndorf erfolgte am 12. Juni 1873. Der Bau wurde dem Lichtensteiner Baumeister Carl Reichenbach übertragen, der auch die Glocke für den Turm beschaffte. Die neue Schule erhielt im Erdgeschoss zwei Klassenzimmer und im Obergeschoss zwei Lehrerwohnungen. Am 17. November 1873 wurde das Gebäude feierlich eingeweiht. Die Schüler, geschmückt mit Stäben und Kränzen, zogen unter Glockengeläut vom alten Schulhaus zur neuen Schule. Der Bau kostete 8795 Taler. {{% /timeline/item %}} {{% timeline/item to="1878-01-01" %}} Die Schülerzahl war auf 154 Kinder angestiegen, sodass die Einstellung eines Hilfslehrers erforderlich wurde. Die Gemeinde zögerte zunächst, entschied sich dann aber für die Anstellung von Hilfslehrer Gerstenberger. Damit wurde die Schule vierklassig. {{% /timeline/item %}} {{% timeline/item to="1884-01-01" %}} Die Anzahl der Schulkinder wuchs weiter und erreichte 300. Es musste ein Seitenflügel mit zwei zusätzlichen Klassenzimmern und einer Lehrerwohnung angebaut werden. Zudem wurde ein dritter Lehrer, Vikar Götze, angestellt. {{% /timeline/item %}} {{% timeline/item to="1888-01-01" %}} Erneuter Platzmangel führte dazu, dass einige versetzungsfähige Kinder nicht in die nächsthöhere Klasse aufgenommen werden konnten. Die Planung eines weiteren Anbaus begann, jedoch entschied man sich schließlich für einen vollständigen Neubau an einem neuen Standort. {{% /timeline/item %}} {{% timeline/item to="1891-01-01" %}} Das bisherige Schulgebäude wurde zur Gemeindeverwaltung umfunktioniert. Damit wurde es zum Rathaus von Hohndorf. {{% /timeline/item %}} {{% timeline/item to="1903-01-01" %}} Aufgrund des starken Wachstums der Gemeinde wurde ein neues Schulgebäude errichtet. Es umfasste neun Schulzimmer sowie eine Turnhalle. Die Baukosten beliefen sich auf 104.052 Mark. {{% /timeline/item %}} {{% timeline/item to="1932-06-21" %}} Beim Umbau des Rathauses wurde der ursprüngliche Grundstein der Schule aus dem Jahr 1873 entdeckt. Das Gebäude wurde um einen Rundbau für die Spar- und Girokasse erweitert. Auch der Bedarf an Verwaltungsräumen war gestiegen, sodass weitere Anpassungen vorgenommen wurden. {{% /timeline/item %}} {{% timeline/item to="1990-01-01" %}} Zwischen 1990 und 1993 erfolgten umfangreiche Umbau- und Reparaturarbeiten am Rathaus. 1994 wurde eine Tiefgarage gebaut. {{% /timeline/item %}} {{% timeline/item to="1998-01-01" %}} Die im Krieg eingezogene Glocke wurde durch eine neue „Friedensglocke“ ersetzt. Der Ratssaal wurde für öffentliche Gemeinderatssitzungen und Eheschließungen genutzt. Zudem begannen regelmäßige Ausstellungen im Treppenhaus des Rathauses. {{% /timeline/item %}} {{< /timeline/timeline >}}
