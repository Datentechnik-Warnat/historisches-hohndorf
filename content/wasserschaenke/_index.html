Die Geschichte der „Wasserschänke“ – Vom Rastplatz zum Einkaufsmarkt

Die „Wasserschänke“ war einst ein beliebter Treffpunkt in Hohndorf, dessen Name bei vielen älteren Einwohnern nostalgische Erinnerungen an rauschende Bälle, Konzerte und fröhliche Tanzabende weckt. Ihre Geschichte beginnt als Raststation für Kohlenfuhrwerke und endet mit einem modernen Einkaufsmarkt. {{< timeline/timeline >}} {{% timeline/item to="1824-01-01" %}} In der Nähe eines Verkehrsknotenpunkts entsteht ein erstes Wirtshaus. Dort kreuzten sich die wichtige Verbindungsstraße von Zwickau über Lichtenstein nach Stollberg sowie die sogenannte Garnstraße - ein alter Hohlweg von Neuschönburg über den Fleischerberg in Richtung Hohenstein-Ernstthal bzw. Waldenburg.
Der ursprüngliche Betreiber stellte einen Antrag auf Schankerlaubnis, doch diese wurde ihm sieben Jahre lang verweigert.
Grund dafür war die Befürchtung, „schlechtes Gesindel“ könnte dort Unterschlupf finden. {{% /timeline/item %}} {{% timeline/item to="1831-01-01" %}} Die Schankgenehmigung wird schließlich auf Druck von Kohlenfuhrleuten aus Reinsdorf erteilt. Diese hatten ein großes Interesse daran, dort Halt zu machen, um mit dem klaren Röhrenwasser ihre Pferde zu tränken. Damit beginnt die offizielle Geschichte der späteren „Wasserschänke“ als Gasthof. {{% /timeline/item %}} {{% timeline/item to="1873-01-01" %}} Der Steinkohlenbauverein Bockwa Vereinigtfeld Hohndorf bei Lichtenstein erwirbt das Gasthaus. Damit beginnt eine engere Verbindung zur regionalen Bergbauindustrie. {{% /timeline/item %}} {{% timeline/item to="1897-01-01" %}} Louis Wagner wird als Pächter der Wasserschänke eingesetzt. Unter seiner Leitung erlebt das Gasthaus eine Blütezeit und entwickelt sich zu einem kulturellen Mittelpunkt des Ortes. {{< imagerow >}} {{< image src="images/grosses_event.jpg" height="280" >}} {{< image src="images/img167_small.jpg" height="280" >}} {{< /imagerow >}} {{% /timeline/item %}} {{% timeline/item to="1903-05-20" %}} Louis Wagner stellt einen Bauantrag zur Errichtung eines Konzert- und Tanzsaals. Die Chemnitzer Firmen Klinger und Schubert übernehmen den Bau. Der fertige Saal wird als der schönste der ganzen Umgebung gefeiert. {{< image src="images/saal.jpg" height="280" >}} {{% /timeline/item %}} {{% timeline/item to="1931-01-01" %}} Zum 70. Geburtstag des Wirts Louis Wagner wird zugleich das 100-jährige Bestehen des Gasthofs gefeiert. Der Gasthof umfasst zu dieser Zeit eine Gaststube, einen kleinen Saal, eine Kegelbahn, Vereinszimmer und einen Garten. Auch die Hohndorfer Schule nutzt den großen Saal für ihre Märchenaufführungen. {{< image src="images/DSC00268_small.jpg" height="280" >}} {{% /timeline/item %}} {{% timeline/item to="1939-09-01" %}} Während des Zweiten Weltkriegs dient der große Saal als Unterkunft für Flüchtlinge. Die eigentliche Nutzung als Veranstaltungsort tritt in den Hintergrund. {{% /timeline/item %}} {{% timeline/item to="1945-01-01" %}} Nach Kriegsende beginnt der Stern der Wasserschänke zu sinken. Kurt Wagner übernimmt den Betrieb von seinem Vater Louis. Zwar gibt es noch Veranstaltungen, doch der Glanz der Vorkriegszeit ist verflogen. {{% /timeline/item %}} {{% timeline/item to="1958-01-01" %}} Der Gaststättenbetrieb wird endgültig eingestellt. Der große Saal dient fortan als Möbellager des Konsums, im Keller werden Kartoffeln eingelagert. {{% /timeline/item %}} {{% timeline/item to="1962-01-01" %}} Das Wohn- und Wirtschaftsgebäude der Wasserschänke wird abgerissen. Damit verschwindet ein bedeutendes Stück Hohndorfer Geschichte aus dem Ortsbild. {{% /timeline/item %}} {{% timeline/item to="1963-01-01" %}} Die Firma A.L. Günther nutzt ab diesem Jahr den großen und kleinen Saal als Weberei. Auch nach der Wende 1990 bleibt das Gebäude in gewerblicher Nutzung durch verschiedene Textilunternehmen. {{% /timeline/item %}} {{% timeline/item to="1992-03-01" %}} {{< image src="images/1992_small.jpg" height="280" title="Die Wasserschänke im Jahr 1992">}} {{% /timeline/item %}} {{% timeline/item to="2010-03-01" %}} Der Abriss der verbliebenen alten Gemäuer der einstigen Wasserschänke beginnt. Damit endet endgültig die bauliche Geschichte des traditionsreichen Ortes.

An der Stelle der früheren Wasserschänke wird der neue NETTO-Einkaufsmarkt feierlich eröffnet. Der obere Teil von Hohndorf erhält dadurch eine moderne Einkaufsmöglichkeit in zentraler Lage. {{% /timeline/item %}} {{< /timeline/timeline >}} {{< nav >}}
